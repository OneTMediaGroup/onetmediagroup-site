# Factory On Call — Production Display Requested By Cleanup

## Change
Removed the **Requested By** field from the live Production Display active call cards.

## Reason
Factory On Call live calls are station-driven. The production display should not imply that an individual operator personally requested the call.

## Production Display now focuses on
- Station
- Personnel Required
- Location
- Waiting
- Status

- `display.js`
- `display.css`
- `display.html`
